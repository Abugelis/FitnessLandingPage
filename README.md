# Fitness Landing Page

Live Demo:

A responsive landing page built with HTML & CSS, focusing on
clean layout, horizontal scrolling sections, and mobile-first design.

## Features
- Responsive layout across devices
- Horizontally scrollable feature & testimonial sections
- Horizontal click & drag to scroll for horizontal cards (desktop only)
- Roadmap-style content layout
- Clean, accessible UI

## Tech Stack
- HTML
- CSS (Flexbox)
- JS

## Notes

Project ran through google PageSpeed Insight (google lighthouse) and
scored 100 in all mobile & desktop sections
